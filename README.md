## 面试重点

1. 简历？把人家列的重要的技术点都问了
2. 面试看重的点？编码能力，逻辑能力，深度广度，学习能力
3. 看人？沟通协调能力、友善
4. 某些问题可以有不同答案，有些问题可以根据别人的回答续继拓展聊下去。发掘别人的优点、找到对方擅长的地方
5. 补充：解决问题、思考的能力，没有答案的问题如何解决？
6. 一定要问原理、底层、思想，为什么有这个技术，要做这些事情，应用场景是什么？或者问他研究最深，最精通点？
